# 🌀 DEEP?

저희 DEE:P은 **명함의 불편함**에서부터 출발했습니다.

현재 사람들이 대중적으로 사용하는 명함은

1. **여러개를 갖고 다녀야 해** 불편해요.
2. 물에 **젖을 가능성**이 있어요.
3. 받은 명함들을 **관리하기 힘들어요.**

그래서 저희는 NFC 명함 스티커라는 색다른 방식의 명함 서비스, **DEE:P**을 개발하게 되었습니다.
